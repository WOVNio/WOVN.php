Install an run phpunit tests 
  - Download the phpunit archive:
    # curl -O https://phar.phpunit.de/phpunit.phar
    # chmod +x phpunit.phar
  - Add the alias to your .bash_profile:
    # vim ~/.bash_profile
    alias phpunit='PATH/TO/phpunit.phar'
    # source ~/.bash_profile
  - Run the tests:
    # cd backend/php/wovnphp/test
    # phpunit .
